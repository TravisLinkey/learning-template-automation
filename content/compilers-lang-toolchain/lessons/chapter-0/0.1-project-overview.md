# 0.1 — Project 5: Language Toolchain

Build a complete language toolchain that combines your terminal emulator and JSON parser into a usable command-line tool.

## Objective

Create a command-line tool that:
1. Uses your PTY terminal emulator for interactive I/O
2. Parses JSON commands using your JSON parser
3. Executes commands and returns results
4. Provides a REPL (Read-Eval-Print Loop) experience

## Architecture

```
┌─────────────────────────────────────────────────────────────┐
│                    Language Toolchain                        │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  User Input (terminal) ──┐                                   │
│                           ▼                                   │
│  ┌─────────────────────────────────────────────────────┐    │
│  │              Terminal Emulator (PTY)                 │    │
│  │    (handles raw input, display output)              │    │
│  └──────────────────────┬──────────────────────────────┘    │
│                          │                                    │
│                          ▼                                    │
│  ┌─────────────────────────────────────────────────────┐    │
│  │              JSON Parser / Command Handler           │    │
│  │    (parses {"cmd": "echo", "args": {...}})          │    │
│  └──────────────────────┬──────────────────────────────┘    │
│                          │                                    │
│                          ▼                                    │
│  ┌─────────────────────────────────────────────────────┐    │
│  │                   Command Executor                   │    │
│  │    (echo, add, parse, evaluate, help, etc.)         │    │
│  └─────────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────┘
```

## Command Format

All commands are JSON objects:

```json
{"cmd": "command_name", "args": {...}}
```

### Available Commands

| Command | Description | Example |
|---------|-------------|---------|
| `echo` | Print a message | `{"cmd": "echo", "args": {"message": "Hello"}}` |
| `add` | Add numbers | `{"cmd": "add", "args": {"a": 5, "b": 3}}` |
| `sub` | Subtract numbers | `{"cmd": "sub", "args": {"a": 10, "b": 3}}` |
| `mul` | Multiply numbers | `{"cmd": "mul", "args": {"a": 4, "b": 5}}` |
| `div` | Divide numbers | `{"cmd": "div", "args": {"a": 10, "b": 2}}` |
| `parse` | Parse JSON | `{"cmd": "parse", "args": {"json": "{\"x\":1}"}}` |
| `evaluate` | Evaluate expression | `{"cmd": "evaluate", "args": {"expr": "1+2*3"}}` |
| `help` | Show help | `{"cmd": "help"}` |
| `exit` | Exit the tool | `{"cmd": "exit"}` |

## Steps to Complete the Project

### 1. Combine projects
Integrate your terminal emulator and JSON parser into one project.

**Hints:**
- Add JSON parser source files to terminal emulator project
- Or link them together in Makefile
- Test that both work independently first

### 2. Command dispatcher
Parse JSON command and route to handler.

**Hints:**
- Extract "cmd" field from parsed JSON
- Look up command in dispatch table
- Pass "args" to handler function

### 3. Implement command handlers
Create functions for each command.

**Hints:**
- `echo` — print message arg
- `add/sub/mul/div` — arithmetic operations
- `parse` — test JSON parser
- `help` — list available commands

### 4. REPL loop
Main loop that reads, parses, executes, prints.

**Hints:**
- Use terminal emulator's main loop as base
- Read JSON input from user
- Parse and execute command
- Print result or error

### 5. Error handling
Handle JSON parse errors and command errors.

**Hints:**
- Catch parser errors, print friendly message
- Unknown commands should show help
- Division by zero handling
