# 1.1 â€” Sample Spaces and Events

Probability theory begins with understanding what can happen and what we're interested in. This lesson introduces the fundamental concepts of **sample spaces** and **events**, which form the foundation for all probability calculations.

## What is a Sample Space?

A **sample space**, denoted by $\Omega$ (omega), is the set of all possible outcomes of a random experiment. Learn more: [Sample Space](https://en.wikipedia.org/wiki/Sample_space)

**Key Points:**
- Each outcome is a single element of the sample space
- The sample space must be exhaustive (cover all possibilities)
- Outcomes must be mutually exclusive (only one can occur)

### Examples of Sample Spaces

**Example 1: Coin Toss**
- Sample space: $\Omega = \{H, T\}$ where H = heads, T = tails
- Two possible outcomes

**Example 2: Rolling a Die**
- Sample space: $\Omega = \{1, 2, 3, 4, 5, 6\}$
- Six possible outcomes

**Example 3: Stock Price Movement**
- Sample space: $\Omega = \{\text{up}, \text{down}, \text{unchanged}\}$
- Three possible outcomes for a single trading day

**Example 4: Portfolio Return**
- Sample space: $\Omega = (-\infty, +\infty)$ (all real numbers)
- Continuous sample space for percentage returns

## Types of Sample Spaces

### Discrete Sample Spaces

A **discrete sample space** has a finite or countably infinite number of outcomes.

- **Finite**: Coin toss ($\{H, T\}$), die roll ($\{1, 2, 3, 4, 5, 6\}$)
- **Countably Infinite**: Number of trades in a day ($\{0, 1, 2, 3, ...\}$)

### Continuous Sample Spaces

A **continuous sample space** has uncountably infinite outcomes, typically intervals of real numbers.

- Stock price: $[0, \infty)$
- Portfolio return: $(-\infty, +\infty)$
- Time to execution: $[0, \infty)$

## What is an Event?

An **event** is a subset of the sample space. An event occurs if the outcome of the experiment is in that subset. Learn more: [Event (Probability Theory)](https://en.wikipedia.org/wiki/Event_(probability_theory))

**Notation:** Events are typically denoted by capital letters: $A$, $B$, $C$, etc.

### Examples of Events

**Example 1: Die Roll**
- Sample space: $\Omega = \{1, 2, 3, 4, 5, 6\}$
- Event $A$ = "Roll an even number" = $\{2, 4, 6\}$
- Event $B$ = "Roll a number greater than 4" = $\{5, 6\}$
- Event $C$ = "Roll a 6" = $\{6\}$ (simple event)

**Example 2: Stock Price**
- Sample space: $\Omega = \{\text{up}, \text{down}, \text{unchanged}\}$
- Event $A$ = "Price increases" = $\{\text{up}\}$
- Event $B$ = "Price doesn't decrease" = $\{\text{up}, \text{unchanged}\}$

**Example 3: Portfolio Return**
- Sample space: $\Omega = (-\infty, +\infty)$
- Event $A$ = "Positive return" = $(0, +\infty)$
- Event $B$ = "Return between -5% and +5%" = $(-0.05, 0.05)$

## Types of Events

### Simple Events

A **simple event** (or elementary event) contains exactly one outcome.

- Rolling a 6: $\{6\}$
- Coin showing heads: $\{H\}$

### Compound Events

A **compound event** contains more than one outcome.

- Rolling an even number: $\{2, 4, 6\}$
- Stock price up or unchanged: $\{\text{up}, \text{unchanged}\}$

### Certain Event

The **certain event** is the entire sample space $\Omega$. It always occurs.

- "Roll a number between 1 and 6" = $\{1, 2, 3, 4, 5, 6\}$ = $\Omega$

### Impossible Event

The **impossible event** is the empty set $\emptyset$. It never occurs.

- "Roll a 7" = $\emptyset$ (for a standard die)

## Set Operations on Events

Since events are sets, we can use set operations:

### Union ($A \cup B$)

The union of events $A$ and $B$ occurs if **either** $A$ **or** $B$ (or both) occurs.

**Definition:** Union ($A \cup B$): All elements in $A$ or $B$ (or both).

**Example:**

$$\{1, 2, 3\} \cup \{3, 4, 5\} = \{1, 2, 3, 4, 5\}$$

### Intersection ($A \cap B$)

The intersection of events $A$ and $B$ occurs if **both** $A$ **and** $B$ occur.

**Example:**
- $A$ = "Roll an even number" = $\{2, 4, 6\}$
- $B$ = "Roll a number greater than 4" = $\{5, 6\}$

$$A \cap B = \{6\}$$

### Complement ($A^c$ or $\bar{A}$)

The complement of event $A$ occurs if $A$ does **not** occur.

**Example:**
- $A$ = "Roll an even number" = $\{2, 4, 6\}$

$$A^c = \{1, 3, 5\}$$

### Mutually Exclusive Events

Events $A$ and $B$ are **mutually exclusive** (or disjoint) if $A \cap B = \emptyset$.

They cannot both occur simultaneously.

**Example:**
- $A$ = "Roll a 1" = $\{1\}$
- $B$ = "Roll a 6" = $\{6\}$
- $A \cap B = \emptyset$ (mutually exclusive)

## Financial Applications

### Portfolio Analysis

**Sample Space:** All possible portfolio returns
- Event $A$: "Portfolio return > 10%"
- Event $B$: "Portfolio return < -5%"
- Event $A \cup B$: "Portfolio return > 10% OR < -5%" (extreme outcomes)

### Risk Management

**Sample Space:** All possible loss scenarios
- Event $A$: "Loss exceeds VaR threshold"
- Event $B$: "Loss exceeds stress test limit"
- Event $A \cap B$: "Loss exceeds both thresholds"

### Trading Signals

**Sample Space:** All possible market conditions
- Event $A$: "Bull market"
- Event $B$: "High volatility"
- Event $A \cap B$: "Bull market with high volatility"

## Python Example

Let's represent sample spaces and events in Python:

```python
import numpy as np

# Sample space for die roll
sample_space = {1, 2, 3, 4, 5, 6}

# Events
event_even = {2, 4, 6}  # Even numbers
event_greater_4 = {5, 6}  # Numbers greater than 4

# Set operations
union = event_even | event_greater_4  # {2, 4, 5, 6}
intersection = event_even & event_greater_4  # {6}
complement_even = sample_space - event_even  # {1, 3, 5}

print(f"Union: {union}")
print(f"Intersection: {intersection}")
print(f"Complement of even: {complement_even}")

# Check if events are mutually exclusive
are_mutually_exclusive = len(event_even & event_greater_4) == 0
print(f"Are mutually exclusive? {are_mutually_exclusive}")
```

## Key Takeaways

1. **Sample Space** ($\Omega$): The set of all possible outcomes
2. **Event**: A subset of the sample space
3. **Simple Event**: Contains exactly one outcome
4. **Compound Event**: Contains multiple outcomes
5. **Set Operations**: Union (or), Intersection (and), Complement (not)
6. **Mutually Exclusive**: Events that cannot occur simultaneously
