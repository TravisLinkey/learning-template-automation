# C Programming

Learn C programming for systems development, building toward a terminal emulator project.

**Target Audience:** Developers wanting to understand systems programming and build low-level projects  
**Skill Progression:** Beginner → Intermediate → Advanced  
**Prerequisite:** Basic programming experience (any language)

---

## Research Findings

### Source Material Analyzed
1. **The C Programming Language** (K&R) — Classic reference
2. **C Programming Absolute Beginner's Guide** — Accessible introduction
3. **Modern C** (Jens Gustedt) — Contemporary C best practices
4. **Linux Programming by Example** — Systems programming on Linux
5. **Unix Programming FAQs** — Practical systems concepts

### Key Concepts to Cover
1. **C Fundamentals** — Syntax, types, control flow
2. **Pointers** — The heart of C, memory addresses, pointer arithmetic
3. **Memory Management** — Stack vs heap, malloc/free, common pitfalls
4. **File I/O** — File descriptors, streams, buffering
5. **Process Control** — fork, exec, wait, signals
6. **PTY/Terminal** — Pseudo-terminals, terminal control
