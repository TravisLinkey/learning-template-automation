# 1.3 — Dynamic Memory (malloc/free)

Static arrays have fixed sizes determined at compile time. Dynamic memory lets you allocate space at runtime, essential for data structures, file handling, and systems programming.

## The Heap vs Stack

```
Memory Layout:
┌─────────────────────┐ high addresses
│       Stack         │ ← local variables, function calls
├─────────────────────┤
│         ↓           │
│         ↑           │
├─────────────────────┤
│       Heap          │ ← dynamic allocations (malloc)
└─────────────────────┘ low addresses
```

- **Stack** — Fast, automatic cleanup, fixed size
- **Heap** — Slower, manual management, flexible size

## malloc - Allocate Memory

```c
#include <stdlib.h>

void *malloc(size_t size);
```

Allocate raw bytes on the heap:

```c
int *ptr = malloc(sizeof(int));  // Allocate 4 bytes

if (ptr == NULL) {
    // Handle allocation failure
    return 1;
}

// Use the memory
*ptr = 42;
printf("%d\n", *ptr);

// Always free when done
free(ptr);
ptr = NULL;  // Good practice
```

**Always check if malloc returns NULL!**

## calloc - Allocate and Zero

```c
void *calloc(size_t nmemb, size_t size);
```

Allocates and zeros memory:

```c
// Allocate array of 10 ints, initialized to 0
int *arr = calloc(10, sizeof(int));

// Equivalent to:
int *arr = malloc(10 * sizeof(int));
memset(arr, 0, 10 * sizeof(int));
```

## realloc - Resize Allocation

```c
void *realloc(void *ptr, size_t size);
```

Resize existing allocation:

```c
int *arr = malloc(5 * sizeof(int));
// ... use arr ...

// Need more space
int *new_arr = realloc(arr, 10 * sizeof(int));
if (new_arr == NULL) {
    free(arr);  // Original still valid, free it
    return 1;
}
arr = new_arr;  // Use new pointer
```

## free - Release Memory

```c
void free(void *ptr);
```

Release allocated memory:

```c
int *ptr = malloc(sizeof(int));
free(ptr);
ptr = NULL;  // Prevent dangling pointer
```

**Rules:**
1. Free only memory you allocated
2. Don't free the same memory twice (double-free)
3. Don't use memory after freeing (dangling pointer)

## Common Patterns

### Allocate Array Dynamically

```c
int size = 100;
int *arr = malloc(size * sizeof(int));

if (arr == NULL) {
    fprintf(stderr, "Allocation failed\n");
    return 1;
}

// Use array
arr[0] = 42;

// Clean up
free(arr);
```

### Resize Array (Growing)

```c
int *arr = NULL;
int size = 0;
int capacity = 10;

// Initial allocation
arr = malloc(capacity * sizeof(int));
if (arr == NULL) return 1;
size = 10;

// Need to grow
if (size >= capacity) {
    capacity *= 2;
    int *new = realloc(arr, capacity * sizeof(int));
    if (new == NULL) {
        free(arr);
        return 1;
    }
    arr = new;
}
```

### Allocate String

```c
char *str = malloc(256);
if (str == NULL) return 1;

// Copy string
strcpy(str, "Hello, World!");

// Use string
printf("%s\n", str);

// Clean up
free(str);
```

## Memory Leak Example

```c
void leak_example(void) {
    int *ptr = malloc(sizeof(int));
    // ... some code ...
    // Forgot to free!
    return;  // Memory leaked!
}
```

## Valgrind for Memory Checking

```bash
# Compile with debug symbols
gcc -g myprogram.c -o myprogram

# Run under valgrind
valgrind --leak-check=full ./myprogram
```

Valgrind reports:
- Memory leaks
- Invalid memory accesses
- Use of uninitialized memory
- Double-free errors

## Common Mistakes

### 1. Forgetting to check NULL
```c
int *ptr = malloc(sizeof(int));
*ptr = 42;  // CRASH if malloc failed!
```

**Fix:** Always check:
```c
int *ptr = malloc(sizeof(int));
if (ptr == NULL) {
    // Handle error
}
```

### 2. Forgetting to free
```c
void process_data(int size) {
    int *data = malloc(size * sizeof(int));
    // ... process ...
    // Forgot free!
}
```

### 3. Double free
```c
int *ptr = malloc(sizeof(int));
free(ptr);
free(ptr);  // CRASH!
```

### 4. Using after free
```c
int *ptr = malloc(sizeof(int));
*ptr = 42;
free(ptr);
printf("%d\n", *ptr);  // CRASH! (undefined behavior)
```

### 5. Wrong size in malloc
```c
// Want array of 10 ints
int *arr = malloc(10);  // WRONG: only 10 bytes!

int *arr = malloc(10 * sizeof(int));  // CORRECT: 40 bytes
```
