# 1.x — Chapter 1 Summary and Quiz

## Chapter Summary

Chapter 1 covered the heart of C programming: pointers and memory management.

### Key Concepts

1. **Pointers and Addresses**
   - Pointers store memory addresses
   - Use `&` to get address, `*` to dereference
   - Pass to functions to modify original values

2. **Arrays and Strings**
   - Arrays decay to pointers
   - Pointer arithmetic moves by element size
   - Strings are null-terminated char arrays

3. **Dynamic Memory**
   - `malloc` allocates raw memory
   - `calloc` allocates and zeros
   - `realloc` resizes
   - `free` releases

4. **Memory Management Patterns**
   - Check all allocations for NULL
   - Pair every malloc with free
   - Set pointers to NULL after freeing

## Question 1: Pointer Basics

**Task:** Write a function `swap` that swaps two integers using pointers.

**Input:**
```c
int a = 5, b = 10;
swap(&a, &b);
// Expected: a = 10, b = 5
```

**Expected Output:**
```
Before: a=5, b=10
After: a=10, b=5
```

<details>
<summary>Show Hint</summary>

Use a temporary variable to swap the values at the addresses.

</details>

<details>
<summary>Show Solution</summary>

```c
void swap(int *x, int *y) {
    int temp = *x;
    *x = *y;
    *y = temp;
}

int main(void) {
    int a = 5, b = 10;
    printf("Before: a=%d, b=%d\n", a, b);
    swap(&a, &b);
    printf("After: a=%d, b=%d\n", a, b);
    return 0;
}
```

</details>

## Question 2: Array-Pointer Relationship

**Task:** What is the output of this code?

```c
#include <stdio.h>

int main(void) {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;
    
    printf("%d\n", *(ptr + 2));
    printf("%d\n", ptr[3]);
    printf("%d\n", *(arr + 1));
    
    return 0;
}
```

**Expected Output:**
```
30
40
20
```

<details>
<summary>Show Solution</summary>

- `*(ptr + 2)` = arr[2] = 30
- `ptr[3]` = arr[3] = 40
- `*(arr + 1)` = arr[1] = 20

Output:
```
30
40
20
```

</details>

## Question 3: Dynamic Memory

**Task:** Write a function that creates a dynamic array of integers with a given size.

**Input:**
```c
int *arr = create_array(5);
// Fill with values 1, 2, 3, 4, 5
```

**Expected Output:**
```
1 2 3 4 5
```

<details>
<summary>Show Hint</summary>

Use malloc to allocate memory, check for NULL, then initialize the array.

</details>

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>
#include <stdlib.h>

int *create_array(size_t size) {
    int *arr = malloc(size * sizeof(int));
    if (arr == NULL) return NULL;
    
    for (size_t i = 0; i < size; i++) {
        arr[i] = (int)(i + 1);
    }
    return arr;
}

int main(void) {
    int *arr = create_array(5);
    if (arr == NULL) return 1;
    
    for (int i = 0; i < 5; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    
    free(arr);
    return 0;
}
```

</details>

## Question 4: String Duplication

**Task:** Write a function that duplicates a string using malloc.

**Input:**
```c
char *copy = strdup("Hello");
// Expected: copy points to new memory containing "Hello"
```

**Expected Output:**
```
Original: Hello
Duplicate: Hello
Addresses differ: (different addresses)
```

<details>
<summary>Show Hint</summary>

Allocate memory for the string length + 1 (for null terminator), then use strcpy.

</details>

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>
#include <string.h>
#include <stdlib.h>

char *strdup(const char *s) {
    size_t len = strlen(s) + 1;
    char *copy = malloc(len);
    if (copy == NULL) return NULL;
    return strcpy(copy, s);
}

int main(void) {
    char *original = "Hello";
    char *duplicate = strdup(original);
    
    printf("Original: %s\n", original);
    printf("Duplicate: %s\n", duplicate);
    printf("Addresses differ: %p != %p\n", 
           (void*)original, (void*)duplicate);
    
    free(duplicate);
    return 0;
}
```

</details>

## Question 5: Memory Debugging

**Task:** Identify the bugs in this code and fix them.

```c
#include <stdio.h>
#include <stdlib.h>

int *create_array_wrong(size_t size) {
    int *arr = malloc(size);
    for (size_t i = 0; i < size; i++) {
        arr[i] = i;
    }
    return arr;
}

int main(void) {
    int *arr = create_array_wrong(10);
    printf("First: %d\n", arr[0]);
    return 0;
}
```

**Expected Output:**
```
First: 0
```

<details>
<summary>Show Hint</summary>

What is `malloc(size)` allocating? How many bytes is that?

</details>

<details>
<summary>Show Solution</summary>

**Bug:** `malloc(size)` allocates only `size` bytes, not `size * sizeof(int)`!

For 10 ints, this allocates only 10 bytes, but we need 40 bytes.

**Fixed code:**
```c
int *create_array_fixed(size_t size) {
    int *arr = malloc(size * sizeof(int));  // Fixed!
    if (arr == NULL) return NULL;
    for (size_t i = 0; i < size; i++) {
        arr[i] = i;
    }
    return arr;
}
```

Also should add free in main:
```c
int main(void) {
    int *arr = create_array_fixed(10);
    printf("First: %d\n", arr[0]);
    free(arr);  // Added!
    return 0;
}
```

</details>
