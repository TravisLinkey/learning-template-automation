## Question 1: File Copy

**Task:** Write a program that copies a file using binary I/O (fread/fwrite) for efficiency.

**Input:** Source filename and destination filename as arguments

**Expected Output:**
```
Copied source.txt to dest.txt
```

<details>
<summary>Show Hint</summary>

Use a buffer to read chunks of the file, then write those chunks to the destination.

</details>

> **Review This Lesson:** [2.1 — File Operations](/systems-c-programming/lessons/2.1)

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>
#include <stdlib.h>

#define BUFFER_SIZE 4096

int main(int argc, char *argv[]) {
    if (argc != 3) {
        fprintf(stderr, "Usage: %s <source> <dest>\n", argv[0]);
        return 1;
    }
    
    FILE *src = fopen(argv[1], "rb");
    if (src == NULL) {
        perror("Failed to open source");
        return 1;
    }
    
    FILE *dst = fopen(argv[2], "wb");
    if (dst == NULL) {
        perror("Failed to open destination");
        fclose(src);
        return 1;
    }
    
    char buffer[BUFFER_SIZE];
    size_t n;
    while ((n = fread(buffer, 1, BUFFER_SIZE, src)) > 0) {
        fwrite(buffer, 1, n, dst);
    }
    
    fclose(src);
    fclose(dst);
    
    printf("Copied %s to %s\n", argv[1], argv[2]);
    return 0;
}
```

</details>

## Question 2: Process Fork

**Task:** What is the output of this code?

```c
#include <stdio.h>
#include <unistd.h>

int main(void) {
    printf("Before fork\n");
    
    pid_t pid = fork();
    
    if (pid == 0) {
        printf("I am the child (PID: %d)\n", getpid());
    } else {
        printf("I am the parent (PID: %d, child: %d)\n", getpid(), pid);
    }
    
    return 0;
}
```

<details>
<summary>Show Hint</summary>

Both parent and child execute the code after fork(). Also note that printf buffering may cause duplicate output.

</details>

> **Review This Lesson:** [2.3 — Process Creation](/systems-c-programming/lessons/2.3)

<details>
<summary>Show Solution</summary>

The output depends on buffering, but typically:
```
Before fork
I am the parent (PID: 1234, child: 1235)
I am the child (PID: 1235)
```

Or with line buffering issue:
```
Before fork
I am the child (PID: 1235)
I am the parent (PID: 1234, child: 1235)
```

Note: "Before fork" might print once or twice depending on buffering.

</details>

## Question 3: Simple Command Execution

**Task:** Write a program that executes `ls -l` using fork and exec.

**Expected Output:**
```
[Directory listing of current directory]
```

<details>
<summary>Show Hint</summary>

Use fork() to create child, then use execl() in child to run /bin/ls.

</details>

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>
#include <unistd.h>
#include <sys/wait.h>

int main(void) {
    pid_t pid = fork();
    
    if (pid < 0) {
        perror("fork failed");
        return 1;
    }
    
    if (pid == 0) {
        // Child: execute ls -l
        execl("/bin/ls", "ls", "-l", NULL);
        perror("exec failed");  // Only runs if exec fails
        _exit(127);
    } else {
        // Parent: wait for child
        int status;
        waitpid(pid, &status, 0);
        printf("ls exited with status %d\n", WEXITSTATUS(status));
    }
    
    return 0;
}
```

## Question 4: Pipe Communication

**Task:** Write a program where the parent sends a message to the child via a pipe.

**Expected Output:**
```
Parent sending: Hello from parent!
Child received: Hello from parent!
```

<details>
<summary>Show Hint</summary>

Create pipe before fork, close appropriate ends in parent and child.

</details>

> **Review This Lesson:** [2.4 — Inter-Process Communication](/systems-c-programming/lessons/2.4)

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>
#include <unistd.h>
#include <sys/wait.h>
#include <string.h>

int main(void) {
    int pipefd[2];
    pipe(pipefd);
    
    pid_t pid = fork();
    
    if (pid == 0) {
        // Child: read from pipe
        close(pipefd[1]);  // Close write end
        
        char buffer[100];
        ssize_t n = read(pipefd[0], buffer, sizeof(buffer));
        buffer[n] = '\0';
        
        printf("Child received: %s", buffer);
        
        close(pipefd[0]);
        _exit(0);
    } else {
        // Parent: write to pipe
        close(pipefd[0]);  // Close read end
        
        const char *msg = "Hello from parent!\n";
        printf("Parent sending: %s", msg);
        write(pipefd[1], msg, strlen(msg));
        
        close(pipefd[1]);
        wait(NULL);
    }
    
    return 0;
}
```

## Question 5: Redirect stdout

**Task:** Write a program that redirects its stdout to a file, then prints "Hello, World!" to that file.

**Expected Output:**
```
[File 'output.txt' contains "Hello, World!"]
```

<details>
<summary>Show Hint</summary>

Use open() to create a file, then dup2() to redirect stdout to that file descriptor.

</details>

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>
#include <fcntl.h>
#include <unistd.h>

int main(void) {
    // Open file, create if doesn't exist, truncate if exists
    int fd = open("output.txt", O_WRONLY | O_CREAT | O_TRUNC, 0644);
    if (fd < 0) {
        perror("open failed");
        return 1;
    }
    
    // Redirect stdout to file
    if (dup2(fd, STDOUT_FILENO) < 0) {
        perror("dup2 failed");
        return 1;
    }
    
    close(fd);  // Original fd no longer needed
    
    // Now printf goes to file!
    printf("Hello, World!\n");
    
    return 0;
}
```

Check result:
```bash
cat output.txt
# Output: Hello, World!
```
