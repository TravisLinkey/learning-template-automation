# 2.3 — Process Creation (fork/exec)

Processes are the foundation of multitasking. This lesson covers creating child processes, running programs, and managing process state.

## What is a Process?

A process is an instance of a running program. Each process has:
- **PID** — Unique Process ID
- **Memory** — Code, data, stack, heap
- **File descriptors** — Open files, sockets, etc.
- **Parent** — The process that created it (PPID)

## fork() — Create a Process

```c
#include <unistd.h>
pid_t fork(void);
```

`fork()` creates a new process by duplicating the calling process:
- **Parent** — Gets child's PID as return value
- **Child** — Gets 0 as return value
- **Error** — Returns -1

```c
#include <stdio.h>
#include <unistd.h>

int main(void) {
    pid_t pid = fork();
    
    if (pid < 0) {
        perror("fork failed");
        return 1;
    }
    
    if (pid == 0) {
        // Child process
        printf("Child: My PID is %d, parent is %d\n", getpid(), getppid());
    } else {
        // Parent process
        printf("Parent: My PID is %d, child is %d\n", getpid(), pid);
    }
    
    return 0;
}
```

Output (order may vary):
```
Parent: My PID is 1234, child is 1235
Child: My PID is 1235, parent is 1234
```

## Visualizing fork()

```
Before fork():
┌─────────────┐
│   Parent    │

After fork():
┌─────────────┐     ┌─────────────┐
```

## exec() — Replace Process Image

`exec` replaces the current process image with a new program:

```c
#include <unistd.h>

// Various exec variants:
int execl(const char *path, const char *arg, ... /* (char *) NULL */);
int execv(const char *path, char *const argv[]);
int execlp(const char *file, const char *arg, ... /* (char *) NULL */);
int execvp(const char *file, char *const argv[]);
```

### Example: Running ls

```c
#include <stdio.h>
#include <unistd.h>

int main(void) {
    printf("Running ls...\n");
    
    // execl: path, arg0, arg1, ..., NULL
    execl("/bin/ls", "ls", "-l", "-a", NULL);
    
    // If we get here, exec failed
    perror("exec failed");
    return 1;
}
```

## fork() + exec() = Running a New Program

```c
#include <stdio.h>
#include <unistd.h>
#include <sys/wait.h>

int main(void) {
    pid_t pid = fork();
    
    if (pid < 0) {
        perror("fork failed");
        return 1;
    }
    
    if (pid == 0) {
        // Child: run new program
        execl("/bin/ls", "ls", "-l", NULL);
        perror("exec failed");  // Only runs if exec fails
        _exit(1);  // Exit child if exec fails
    } else {
        // Parent: wait for child
        int status;
        waitpid(pid, &status, 0);
        printf("Child exited with status %d\n", WEXITSTATUS(status));
    }
    
    return 0;
}
```

## wait() — Wait for Child

```c
#include <sys/wait.h>

pid_t wait(int *status);
pid_t waitpid(pid_t pid, int *status, int options);
```

Wait for child process to exit and get its status:

```c
int status;
pid_t child_pid = wait(&status);  // Blocks until any child exits

if (WIFEXITED(status)) {
    printf("Child exited with status %d\n", WEXITSTATUS(status));
}
```

### Wait Options

| Option | Description |
|--------|-------------|
| `0` | Block until child exits |
| `WNOHANG` | Don't block, return immediately |
| `WUNTRACED` | Stopped as well as exited |

## Process Lifecycle

```
fork() ──▶ child runs ──▶ exit() ──▶ parent waits ──▶ reap (zombie)
              │                 │
              │            wait() returns
              ▼
          exec() (optional)
```

## Signal Handling (Brief)

Signals notify processes of events:

```c
#include <signal.h>

// Handler function
void handler(int sig) {
    printf("Caught signal %d\n", sig);
}

// Register handler
signal(SIGINT, handler);
```

Common signals:
- `SIGINT` (2) — Interrupt (Ctrl+C)
- `SIGTERM` (15) — Termination request
- `SIGKILL` (9) — Kill (cannot be caught)
- `SIGCHLD` (17) — Child exited

## Complete Example: Simple Shell

```c
#include <stdio.h>
#include <unistd.h>
#include <sys/wait.h>
#include <string.h>

int main(void) {
    char input[256];
    
    while (1) {
        printf("shell> ");
        if (fgets(input, sizeof(input), stdin) == NULL) break;
        
        // Remove newline
        input[strcspn(input, "\n")] = '\0';
        if (strcmp(input, "exit") == 0) break;
        
        pid_t pid = fork();
        
        if (pid < 0) {
            perror("fork failed");
            continue;
        }
        
        if (pid == 0) {
            // Child: parse and execute command
            char *args[64];
            char *token = strtok(input, " ");
            int i = 0;
            while (token != NULL) {
                args[i++] = token;
                token = strtok(NULL, " ");
            }
            args[i] = NULL;
            
            execvp(args[0], args);
            perror("command not found");
            _exit(127);
        } else {
            // Parent: wait for child
            wait(NULL);
        }
    }
    
    return 0;
}
```
