# 0.3 — Basic Syntax and Data Types

This lesson covers the fundamental building blocks of C: variables, data types, operators, and control flow structures.

## Basic Program Structure

```c
#include <stdio.h>

int main(void) {
    // Your code here
    return 0;
}
```

- `#include <stdio.h>` — Header for standard I/O (printf, scanf)
- `int main(void)` — Main function, entry point
- `return 0;` — Exit successfully

## Data Types

### Fundamental Types

| Type | Description | Size (typical) |
|------|-------------|----------------|
| `char` | Single character | 1 byte |
| `int` | Integer | 4 bytes |
| `float` | Floating point | 4 bytes |
| `double` | Double precision | 8 bytes |
| `void` | No type | — |

### Type Modifiers

| Modifier | Description |
|----------|-------------|
| `short` | Shorter integer (2 bytes) |
| `long` | Longer integer (8 bytes on 64-bit) |
| `signed` | Can be negative |
| `unsigned` | Non-negative only |

```c
char c = 'A';
int count = 42;
float price = 19.99;
double precise = 3.14159265358979;
unsigned int age = 25;
```

### Sizeof

Use `sizeof()` to get the size of any type:

```c
printf("int size: %zu\n", sizeof(int));
printf("char size: %zu\n", sizeof(char));
```

## Variables

### Declaration and Initialization

```c
// Declaration
int age;

// Declaration + Initialization
int age = 25;

// Multiple declarations
int x = 1, y = 2, z = 3;
```

### Variable Naming Rules

- Start with letter or underscore
- Can contain letters, digits, underscore
- Case-sensitive (`age` ≠ `Age`)
- Cannot be a C keyword

### Constants

```c
// Compile-time constant
#define MAX_SIZE 100

// Or use const
const int MAX_SIZE = 100;
```

## Operators

### Arithmetic Operators

```c
int a = 10, b = 3;

int sum = a + b;      // 13
int diff = a - b;     // 7
int prod = a * b;     // 30
int quot = a / b;     // 3 (integer division)
int rem = a % b;      // 1 (remainder)
```

### Increment/Decrement

```c
int x = 5;
x++;    // x is now 6 (post-increment)
++x;    // x is now 7 (pre-increment)
x--;    // x is now 6
--x;    // x is now 5
```

### Comparison Operators

```c
// Returns 1 (true) or 0 (false)
5 == 5    // 1
5 != 3    // 1
7 > 3     // 1
2 < 1     // 0
```

### Logical Operators

```c
int a = 1, b = 0;

a && b    // 0 (AND)
a || b    // 1 (OR)
!a       // 0 (NOT)
```

## Control Flow

### If-Else

```c
int age = 18;

if (age >= 18) {
    printf("Adult\n");
} else {
    printf("Minor\n");
}
```

### Else If

```c
int score = 85;

if (score >= 90) {
    printf("A\n");
} else if (score >= 80) {
    printf("B\n");
} else if (score >= 70) {
    printf("C\n");
} else {
    printf("F\n");
}
```

### Ternary Operator

```c
int age = 18;
printf("%s\n", age >= 18 ? "Adult" : "Minor");
```

### Switch

```c
char grade = 'B';

switch (grade) {
    case 'A':
        printf("Excellent\n");
        break;
    case 'B':
        printf("Good\n");
        break;
    case 'C':
        printf("Fair\n");
        break;
    default:
        printf("Unknown\n");
}
```

### While Loop

```c
int i = 0;
while (i < 5) {
    printf("%d\n", i);
    i++;
}
```

### For Loop

```c
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}
```

### Do-While

```c
int i = 0;
do {
    printf("%d\n", i);
    i++;
} while (i < 5);
```

## Input and Output

### Printf

```c
// Basic
printf("Hello, World!\n");

// With variables
int age = 25;
printf("Age: %d\n", age);
printf("Price: %.2f\n", 19.99);

// Format specifiers
%d    // int
%ld   // long int
%f    // float
%lf   // double
%c    // char
%s    // string
%p    // pointer
%%    // literal %
```

### Scanf

```c
int age;
printf("Enter your age: ");
scanf("%d", &age);  // Note: & before variable name
printf("You are %d years old\n", age);
```

## Example Program

```c
#include <stdio.h>

int main(void) {
    int a, b;
    
    printf("Enter two numbers: ");
    scanf("%d %d", &a, &b);
    
    printf("\n%d + %d = %d\n", a, b, a + b);
    printf("%d - %d = %d\n", a, b, a - b);
    printf("%d * %d = %d\n", a, b, a * b);
    printf("%d / %d = %d (quotient)\n", a, b, a / b);
    printf("%d %% %d = %d (remainder)\n", a, b, a % b);
    
    return 0;
}
```
