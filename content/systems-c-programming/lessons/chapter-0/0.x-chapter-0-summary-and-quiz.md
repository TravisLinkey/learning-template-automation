# 0.x — Chapter 0 Summary and Quiz

## Chapter Summary

Chapter 0 covered the foundations of C programming:

### Key Concepts

1. **Why C Matters**
   - Direct memory access, no runtime overhead
   - Foundation for operating systems, embedded systems
   - Compile directly to machine code

2. **Development Environment**
   - GCC/Clang compiler
   - Make for build automation
   - GDB for debugging, Valgrind for memory checking

3. **Basic Syntax**
   - Data types: char, int, float, double
   - Variables and constants
   - Operators: arithmetic, comparison, logical
   - Control flow: if-else, switch, while, for
   - I/O: printf, scanf

### Commands Learned

```bash
# Compile
gcc source.c -o executable

# Compile with warnings
gcc -Wall -Wextra source.c -o executable

# Build with Make
make && make clean
```

## Question 1: Compilation Basics

**Task:** What command compiles `hello.c` into an executable named `hello` with all warnings enabled?

**Input:**
```c
// hello.c
#include <stdio.h>
int main(void) {
    printf("Hello!\n");
    return 0;
}
```

**Expected Output:**
Command that produces working executable with warnings

<details>
<summary>Show Hint</summary>

Use the `-Wall` flag for warnings and `-o` to name the output file.

</details>

<details>
<summary>Show Solution</summary>

```bash
gcc -Wall hello.c -o hello
```

</details>

## Question 2: Data Types

**Task:** Write a C program that declares one variable of each type and prints their sizes using `sizeof()`.

**Types to use:** `char`, `int`, `float`, `double`

**Expected Output:**
```
char size: 1
int size: 4
float size: 4
double size: 8
```
(Note: sizes may vary by system)

<details>
<summary>Show Hint</summary>

Use `sizeof(type)` with printf and the `%zu` format specifier.

</details>

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>

int main(void) {
    printf("char size: %zu\n", sizeof(char));
    printf("int size: %zu\n", sizeof(int));
    printf("float size: %zu\n", sizeof(float));
    printf("double size: %zu\n", sizeof(double));
    return 0;
}
```

</details>

## Question 3: Temperature Converter

**Task:** Write a program that converts Fahrenheit to Celsius.

**Input:**
```
Enter temperature in Fahrenheit: 98.6
```

**Expected Output:**
```
98.6°F = 37.0°C
```

**Formula:** C = (F - 32) * 5 / 9

<details>
<summary>Show Hint</summary>

Use `float` for temperature values. The formula involves division, so use `5.0` or cast to float to avoid integer division.

</details>

<details>
<summary>Show Solution</summary>

```c
#include <stdio.h>

int main(void) {
    float fahrenheit, celsius;
    
    printf("Enter temperature in Fahrenheit: ");
    scanf("%f", &fahrenheit);
    
    celsius = (fahrenheit - 32.0) * 5.0 / 9.0;
    
    printf("%.1f°F = %.1f°C\n", fahrenheit, celsius);
    return 0;
}
```

</details>

## Question 4: Loop Output

**Task:** What is the output of the following code?

```c
#include <stdio.h>

int main(void) {
    for (int i = 1; i <= 5; i++) {
        if (i % 2 == 0) {
            printf("%d is even\n", i);
        } else {
            printf("%d is odd\n", i);
        }
    }
    return 0;
}
```

**Expected Output:**
```
1 is odd
2 is even
3 is odd
4 is even
5 is odd
```

<details>
<summary>Show Solution</summary>

The output is:
```
1 is odd
2 is even
3 is odd
4 is even
5 is odd
```

The loop iterates 1-5, checking if each number is divisible by 2 (even) or not (odd).

</details>

## Question 5: Debug the Code

**Task:** Find and fix the bugs in the following code that calculates the sum of numbers 1 to 10.

```c
#include <stdio.h>

int main(void) {
    int sum = 0;
    int i = 1;
    
    while (i <= 10); {
        sum = sum + i;
        i++;
    }
    
    printf("Sum: %d\n", sum);
    return 0;
}
```

**Expected Output:**
```
Sum: 55
```

<details>
<summary>Show Hint</summary>

Look closely at the while condition line...

</details>

<details>
<summary>Show Solution</summary>

The bug is the semicolon after `while (i <= 10);`. This creates an empty loop that runs infinitely (or until `i` overflows).

Fixed code:
```c
#include <stdio.h>

int main(void) {
    int sum = 0;
    int i = 1;
    
    while (i <= 10) {  // No semicolon after condition
        sum = sum + i;
        i++;
    }
    
    printf("Sum: %d\n", sum);
    return 0;
}
```

Output: `Sum: 55`

</details>
